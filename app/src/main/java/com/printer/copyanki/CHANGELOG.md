# История изменений CopyAnki

## [v1.2.0] - 2026-02-25
### Выезжающая панель статистики и улучшение интерфейса

#### 🎨 НОВАЯ СИСТЕМА НАВИГАЦИИ
1. **Выезжающая панель статистики**
    - Панель появляется слева при нажатии на кнопку
    - Плавная анимация выезда/заезда (300ms)
    - Автоматическое закрытие при взаимодействии с основным контентом
    - Поддержка скролла при большом количестве контента

2. **Инновационная кнопка открытия**
    - Кнопка расположена по центру вертикали слева
    - Вращается только текст внутри (`›` → `<` и обратно)
    - Цвет кнопки совпадает с цветом панели статистики
    - Закруглены только правые края (не соприкасающиеся с панелью)

#### 📊 РЕДИЗАЙН ПАНЕЛИ СТАТИСТИКИ
1. **Новая структура расположения**
    - Статистика параметров вверху (всего слов, изучено, сложные, сегодня)
    - Прогресс обучения в самом низу панели
    - Четкие промежутки между элементами:
        * Большие промежутки (16dp) между основными блоками
        * Средние промежутки (10-12dp) между эмодзи и значением
        * Компактные отступы для визуального разделения

2. **Визуальное оформление**
    - Эмодзи прогресса (📈) и подпись "прогресс" в одной строке
    - Увеличенный прогресс-бар (14px высота)
    - Крупный текст процентов (22sp) под прогресс-баром
    - Единый стиль для всех значений статистики

#### 🖥️ ОПТИМИЗАЦИЯ ПОД РАЗНЫЕ ЭКРАНЫ
1. **Адаптивные размеры**
    - Уменьшены размеры всех элементов с учетом кнопки открытия
    - Кнопка озвучки: 48x48dp
    - Поле ввода: высота 48dp
    - Кнопка подтвердить: высота 48dp
    - Отступ для кнопки: 32dp

2. **Полноэкранный режим**
    - Скрыта системная панель (время, батарея)
    - Увеличенная полезная площадь экрана
    - Поддержка обеих ориентаций

#### 🎯 УЛУЧШЕНИЯ ПОЛЬЗОВАТЕЛЬСКОГО ОПЫТА
1. **Автоматическое закрытие панели**
    - При клике на поле ввода
    - При нажатии кнопки озвучки
    - При подтверждении ответа
    - При клике на пустое место

2. **Плавные анимации**
    - Выезд панели с DecelerateInterpolator
    - Синхронное движение кнопки с панелью
    - Вращение текста без движения кнопки

#### 🐛 ИСПРАВЛЕНИЕ ОШИБОК
1. **Прокрутка статистики**
    - Добавлен ScrollView для контента
    - Скрыты полосы прокрутки для чистоты интерфейса
    - Корректное отображение всех элементов

2. **Отображение прогресса**
    - Проценты теперь всегда видны
    - Правильный порядок элементов
    - Корректные отступы между блоками

## [v1.1.0] - 2026-02-24
### Полная адаптация под вертикальную ориентацию и исправление ошибок

#### 📱 АДАПТАЦИЯ ИНТЕРФЕЙСА ПОД ВЕРТИКАЛЬНЫЙ РЕЖИМ
1. **Полная переработка размеров**
    - Все элементы интерфейса уменьшены пропорционально для компактного отображения
    - Ширина боковых панелей (статистика и управление) уменьшена до 80px
    - Высота верхней панели уменьшена до 48dp
    - Высота нижней панели уменьшена до 32dp
    - Все отступы и padding приведены к минимальным значениям

2. **Оптимизация расположения**
    - Статистика и управление закреплены по краям экрана
    - Центральная панель обучения занимает все оставшееся пространство
    - Версия приложения перемещена в правый нижний угол
    - Прогресс, счетчик и статус звука сгруппированы слева

#### 🎨 УЛУЧШЕНИЯ ВИЗУАЛЬНЫХ КОМПОНЕНТОВ
1. **Кнопки и элементы управления**
    - Кнопка динамика приведена к одному размеру с текстом (28sp)
    - Высота поля ввода увеличена до 60px для комфортного ввода
    - Высота кнопки проверки установлена 56px с правильными отступами
    - Кнопки в панели управления имеют фиксированную высоту 44px

2. **Текстовая информация**
    - Иконка режима отображается в формате "🇺🇸→🇷🇺" (без пробелов)
    - Подсказки сокращены: "через 3", "готова", "активна"
    - Статистика отображается только числами с иконками
    - Прогресс отображается в формате "0%" и "•0"

#### 🔧 ИСПРАВЛЕНИЕ ОШИБОК
1. **Проблемы с ScrollView**
    - Удален ScrollView из stats_panel.xml и control_panel.xml
    - Заменен на LinearLayout с правильным распределением весов
    - Исправлены ошибки приведения типов

2. **Корректное позиционирование**
    - Исправлено наложение кнопок управления друг на друга
    - Статистика больше не уходит в левый угол
    - Версия правильно расположена в правом нижнем углу
    - Центральная панель корректно растягивается между боковыми

3. **Размеры кнопок**
    - Кнопка динамика больше не меньше текста
    - Кнопка проверки отображается полностью с читаемым текстом
    - Все элементы пропорциональны друг другу

#### 📝 ТЕХНИЧЕСКИЕ УЛУЧШЕНИЯ
1. **Переход на XML layouts**
    - Все основные экраны переведены на XML разметку
    - Контроллеры только управляют данными, не создают UI
    - Упрощена поддержка и модификация интерфейса

2. **Оптимизация весовых коэффициентов**
    - stats:control:center = 1.2 : 1.2 : 7.6
    - Правильное распределение пространства на экране
    - Адаптивность под разные размеры экранов

#### 🎯 ПОЛЬЗОВАТЕЛЬСКИЙ ОПЫТ
1. **Улучшенная читаемость**
    - Все элементы хорошо видны на компактном экране
    - Тексты не обрезаются и не накладываются друг на друга
    - Интуитивно понятное расположение элементов

2. **Плавные анимации**
    - Сохранены все анимации успеха/ошибки из оригинальной версии
    - Корректная работа визуальных эффектов

## [v1.0.0] - 2026-02-24
### Первый стабильный релиз для Android

#### 🎉 ПОЛНЫЙ ПЕРЕНОС ФУНКЦИОНАЛА LinguaMaster v1.12.0

#### 📱 ОСНОВНЫЕ ВОЗМОЖНОСТИ
1. **Три режима обучения**
    - ✍️ Ручной перевод с умными подсказками
    - 📝 Тест с выбором варианта
    - 🔄 Соотношение слов (сопоставление)

2. **Умные подсказки**
    - Правильный ответ появляется после заданного числа ошибок
    - Автоматическое исчезновение при начале ввода
    - Восстановление подсказки при пустом поле
    - Визуальная индикация статуса подсказки

3. **Система категорий**
    - Организация слов по категориям
    - Фильтрация по категориям в тренировке
    - Создание новых категорий при добавлении слов

4. **Поддержка 11 языков**
    - 🇺🇸 Английский, 🇷🇺 Русский, 🇪🇸 Испанский, 🇫🇷 Французский
    - 🇩🇪 Немецкий, 🇨🇳 Китайский, 🇯🇵 Японский, 🇰🇷 Корейский
    - 🇵🇹 Португальский, 🇮🇹 Итальянский, 🇸🇦 Арабский

#### 🔊 ОЗВУЧКА
1. **Android TTS (вместо Edge-TTS/RHVoice)**
    - Автоматическое определение языка слова
    - Кнопка динамика рядом со словом
    - Автоматическая озвучка новых слов (опционально)
    - Сохранение настроек голосов

2. **Настройки озвучки**
    - Включение/отключение озвучки
    - Регулировка громкости
    - Регулировка скорости речи
    - Выбор голосов для каждого языка

#### 📊 СТАТИСТИКА
1. **История обучения**
    - Сохранение статистики за каждый день
    - Общее количество слов
    - Количество изученных слов
    - Количество сложных слов
    - Прогресс обучения
    - Время в приложении

2. **Сложные слова**
    - Автоматическое определение сложных слов
    - Отдельный режим тренировки сложных слов
    - Визуальная индикация уровня сложности

#### ⚙️ НАСТРОЙКИ
1. **Основные настройки**
    - Автопереход к следующему слову
    - Включение/отключение подсказок
    - Настройка порога ошибок для подсказок (1-10)
    - Порог сложных слов (0-100%)

2. **Сохранение настроек**
    - Автоматическое сохранение всех параметров
    - Восстановление при следующем запуске
    - Сохранение выбранного языка и категории
    - Сохранение метода обучения

#### 🎨 ИНТЕРФЕЙС
1. **Темная тема**
    - Все цвета сохранены из оригинальной версии
    - Карточки, кнопки, фон в соответствии с дизайном
    - Адаптивный интерфейс

2. **Визуальные эффекты**
    - Анимации успеха/ошибки
    - Эффекты при наведении на кнопки
    - Индикация прогресса
    - Всплывающие уведомления

1. **JSON файлы в internal storage**
    - `vocabulary.json` - словарь пользователя

2. **Автоматическая миграция**
