## [v1.2.1] - 2026-02-26

### Added
- Нижняя панель (BottomBar): отображение версии в формате `© {year} {app_name} {version}`, все 3 значения загружаются из Config.TEXTS (год, название, версия), текст прижат к правому краю нижней панели, размер шрифта 9sp, цвет text_secondary
- Статус озвучки слева внизу (`🔊/🔇`)
- Два прогресс-бара в панели статистики:
  - `📈 общий прогресс` — процент изученных слов от общего числа, синий прогресс-бар (primary color), крупный текст процентов (22sp) под баром
  - `📈 прогресс дня` — точность ответов за сегодня, зеленый прогресс-бар (secondary color), крупный текст процентов (22sp) под баром

### Removed
- Удалён счетчик слов сегодня (`•0`) из нижней панели
- Удалён прогресс из нижней панели

## [v1.2.0] - 2026-02-25

### Added
- Выезжающая панель статистики: панель появляется слева при нажатии на кнопку, плавная анимация выезда/заезда (300ms), автоматическое закрытие при взаимодействии с основным контентом, поддержка скролла при большом количестве контента
- Инновационная кнопка открытия: кнопка расположена по центру вертикали слева, вращается только текст внутри (`›` → `<` и обратно), цвет кнопки совпадает с цветом панели статистики, закруглены только правые края (не соприкасающиеся с панелью)
- Редизайн панели статистики: статистика параметров вверху (всего слов, изучено, сложные, сегодня), прогресс обучения в самом низу панели
- Четкие промежутки между элементами: большие промежутки (16dp) между основными блоками, средние промежутки (10-12dp) между эмодзи и значением, компактные отступы для визуального разделения
- Визуальное оформление: эмодзи прогресса (📈) и подпись "прогресс" в одной строке, увеличенный прогресс-бар (14px высота), крупный текст процентов (22sp) под прогресс-баром, единый стиль для всех значений статистики
- Адаптивные размеры: уменьшены размеры всех элементов с учетом кнопки открытия, кнопка озвучки 48x48dp, поле ввода высота 48dp, кнопка подтвердить высота 48dp, отступ для кнопки 32dp
- Полноэкранный режим: скрыта системная панель (время, батарея), увеличенная полезная площадь экрана, поддержка обеих ориентаций
- Автоматическое закрытие панели: при клике на поле ввода, при нажатии кнопки озвучки, при подтверждении ответа, при клике на пустое место
- Плавные анимации: выезд панели с DecelerateInterpolator, синхронное движение кнопки с панелью, вращение текста без движения кнопки
- Добавлен ScrollView для контента статистики, скрыты полосы прокрутки для чистоты интерфейса

### Fixed
- Проценты теперь всегда видны
- Правильный порядок элементов
- Корректные отступы между блоками
- Корректное отображение всех элементов статистики

- Тексты не обрезаются и не накладываются друг на друга
- Сохранены все анимации успеха/ошибки из оригинальной версии

## [v1.0.0] - 2026-02-24

### Added
- Первый стабильный релиз для Android с полным переносом функционала LinguaMaster v1.12.0
- Три режима обучения: Ручной перевод с умными подсказками, Тест с выбором варианта, Соотношение слов (сопоставление)
- Умные подсказки: правильный ответ появляется после заданного числа ошибок, автоматическое исчезновение при начале ввода, восстановление подсказки при пустом поле, визуальная индикация статуса подсказки
- Система категорий: организация слов по категориям, фильтрация по категориям в тренировке, создание новых категорий при добавлении слов
- Поддержка 11 языков: английский, русский, испанский, французский, немецкий, китайский, японский, корейский, португальский, итальянский, арабский
- Озвучка через Android TTS: автоматическое определение языка слова, кнопка динамика рядом со словом, автоматическая озвучка новых слов (опционально), сохранение настроек голосов
- Настройки озвучки: включение/отключение озвучки, регулировка громкости, регулировка скорости речи, выбор голосов для каждого языка
- История обучения: сохранение статистики за каждый день, общее количество слов, количество изученных слов, количество сложных слов, прогресс обучения, время в приложении
- Сложные слова: автоматическое определение сложных слов, отдельный режим тренировки сложных слов, визуальная индикация уровня сложности
- Основные настройки: автопереход к следующему слову, включение/отключение подсказок, настройка порога ошибок для подсказок (1-10), порог сложных слов (0-100%)
- Сохранение настроек: автоматическое сохранение всех параметров, восстановление при следующем запуске, сохранение выбранного языка и категории, сохранение метода обучения
- Темная тема: все цвета сохранены из оригинальной версии, карточки, кнопки, фон в соответствии с дизайном, адаптивный интерфейс
- Визуальные эффекты: анимации успеха/ошибки, эффекты при наведении на кнопки, индикация прогресса, всплывающие уведомления
- JSON файлы в internal storage: vocabulary.json (словарь пользователя), stats.json (статистика обучения), app_settings.json (настройки приложения)
- Автоматическая миграция: добавление поля category для старых слов, конвертация старой статистики в новый формат, создание файлов при первом запуске
- Архитектура: MVC-подобная архитектура (как в оригинале), Kotlin вместо Python, XML layouts вместо Tkinter, Android TTS вместо Edge-TTS/RHVoice
- Классы и компоненты: MainActivity (точка входа), Config (конфигурация), VocabularyModel (модель данных), MainWindow (главный контроллер), отдельные панели для каждого режима обучения
- Конфликты имен в Kotlin: переименованы методы, конфликтующие со свойствами, исправлены Platform declaration clash, корректная работа с nullable типами
- Ресурсы: созданы недостающие drawable ресурсы, настроены цвета в соответствии с оригиналом, адаптированы размеры для Android
- Минимальные изменения в логике: сохранена вся бизнес-логика из оригинальной версии, изменен только UI-фреймворк (Tkinter → Android Views), адаптирована озвучка под Android TTS
- Использование стандартных Android компонентов вместо кастомных, сохранение JSON для данных (как в оригинале), адаптация размеров под мобильные экраны

### Fixed
- Исправлены конфликты имен в Kotlin
- Исправлены Platform declaration clash
- Исправлена работа с nullable типами
- Созданы недостающие drawable ресурсы
